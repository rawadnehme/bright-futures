# BrightFutures ‚Äî Ground-Up Rebuild Design

**Date**: 2026-04-23
**Status**: Approved (user granted full creative control)

## Concept

"Show Up Ready" ‚Äî the site should feel like it was handmade by the students who run the charity: warm, honest, and a little kinetic, but technically sharp. Animation language borrows from school notebooks: annotations, hand-drawn underlines, sticky-note tilts, paper warmth.

## Brand

- **Amber** `#F5A524` ‚Äî backpack yellow, optimism (primary)
- **Ink** `#0B1220` ‚Äî deep navy chalkboard (text/background)
- **Paper** `#FEFAF1` ‚Äî cream notebook page (alt surface)
- **Green** `#2EB88A` ‚Äî CashApp card, growth
- **Blue** `#3B82F6` ‚Äî Venmo card
- **Typography**: Fraunces (display) + Inter (body), via Google Fonts `<link>` in `index.html` (no JSX `@import`)

## Motion

Every section uses framer-motion. Building blocks live in `src/components/motion/`:

- `Reveal` ‚Äî fade + lift on view (replaces the inline `initial/whileInView` noise)
- `Underline` ‚Äî SVG hand-drawn underline, stroke `pathLength` animates on view
- `Counter` ‚Äî animated number count-up via `useMotionValue` + `animate()`
- `MagneticButton` ‚Äî CTA follows cursor subtly, spring return on leave
- `ScrollProgress` ‚Äî fixed amber bar at top, bound to `useScroll().scrollYProgress`
- `Reveal` + `stagger` container for grids

All motion respects `prefers-reduced-motion` (framer-motion's `useReducedMotion`).

## Page Structure

One page, anchor navigation, no router.

1. **Nav** ‚Äî floating, transitions from transparent (over hero) to cream + backdrop-blur on scroll
2. **Hero** ‚Äî huge Fraunces headline "Every kid shows up. We help them show up *ready.*" Primary CTA (magnetic). Floating SVG school-supply glyphs with gentle parallax.
3. **Mission** ‚Äî "We see it every day." Three sticky-note cards with subtle random tilt (Books, Community, Futures).
4. **Story** ‚Äî split layout, image + text, "Just students, making a difference." Floating badge "Dearborn, MI."
5. **Impact** ‚Äî dark section, 3 animated counters (500+, 100%, ‚àû).
6. **Donate** ‚Äî suggested amount grid + two donation cards (CashApp `$SaintGrav`, Venmo `@MichiganBrightFuture`).
7. **Footer** ‚Äî email, tagline, small print.
